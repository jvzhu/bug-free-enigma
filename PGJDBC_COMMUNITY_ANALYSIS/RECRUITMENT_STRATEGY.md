## 💼 Maintainer Role Description

### Responsibilities

1. **Code Review (40%)**
   - Review PRs in assigned areas
   - Ensure quality standards
   - Provide constructive feedback
   - 2-5 hours/week

2. **Issue Triage (20%)**
   - Categorize new issues
   - Assign to appropriate reviewers
   - Close duplicates/invalid
   - 1-2 hours/week

3. **Feature Development (20%)**
   - Implement features
   - Fix critical bugs
   - 2-3 hours/week

4. **Community (15%)**
   - Mentor contributors
   - Answer questions
   - Participate in discussions
   - 1-2 hours/week

5. **Releases (5%)**
   - Quarterly/as-needed
   - Version management
   - Release notes
   - 2-4 hours per release

**Total Commitment:** 6-15 hours/week (flexible)
