# 🤝 PgJDBC Mentorship Program

> **Objective:** Foster contributor growth and build a sustainable maintainer pipeline  
> **Launch Date:** July 1, 2026  
> **Program Duration:** 12 weeks per cohort

---

## 📋 Program Overview

### Goals

- 🎯 Onboard 15-20 new contributors per quarter
- 🎯 Develop 3-5 future maintainers per year
- 🎯 Reduce time-to-first-contribution from months to weeks
- 🎯 Build sustainable community growth
- 🎯 Increase diversity in maintainer roles

## 📅 12-Week Curriculum

### Week 1-2: Onboarding
- [ ] Set up dev environment
- [ ] Build from source
- [ ] Run test suite
- [ ] Read CONTRIBUTING.md
- [ ] Explore codebase structure

**Deliverable:** Successful local build

### Week 3-4: First Contribution
- [ ] Find "good first issue"
- [ ] Understand issue context
- [ ] Implement fix/feature
- [ ] Run tests locally
- [ ] Submit PR

**Deliverable:** First PR submitted

### Week 5-6: Code Review & Refinement
- [ ] Respond to code review
- [ ] Make requested changes
- [ ] Learn CI/CD system
- [ ] Explore related code

**Deliverable:** First PR merged ✅

### Week 7-8: Feature Development
- [ ] Pick medium-complexity issue
- [ ] Design approach
- [ ] Implementation
- [ ] Test coverage
- [ ] Submit PR

**Deliverable:** 2nd PR submitted

### Week 9-10: Specialization
- [ ] Choose focus area
- [ ] Deep dive into subsystem
- [ ] 2-3 related issues
- [ ] Build expertise

**Deliverable:** 3 PRs submitted (area-focused)

### Week 11-12: Graduation
- [ ] Reflect on learning
- [ ] Identify next steps
- [ ] Commit to continued growth
- [ ] Consider review responsibilities

**Deliverable:** Completion certificate

## 📞 Program Operations

### How to Apply

**As Mentee:**
1. React with 👋 to mentorship issue
2. Comment with motivation & goals
3. Complete questionnaire
4. Wait for mentor matching (1 week)

**As Mentor:**
1. Comment on issue with interest
2. Verify status (2 years/100 commits)
3. Attend orientation
4. Get assigned mentees

### Communication Channels

- **Async:** GitHub issues, PRs, discussions
- **Sync:** Weekly 1:1 calls (30-60 min)
- **Community:** Slack/Discord #mentorship
- **Escalation:** @davecramer (program lead)
