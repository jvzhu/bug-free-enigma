## 🔍 Key Indicators to Watch

### 🟢 Green Lights (Positive)

✅ New contributors > 5/month  
✅ PR merge time < 3 weeks  
✅ 70%+ of issues resolved  
✅ Code coverage > 80%  
✅ 3+ active mentees → active contributors  
✅ GitHub stars growing  
✅ Release on schedule  
✅ Positive community sentiment  

### 🟡 Yellow Lights (Watch)

⚠️ 2-5 new contributors/month  
⚠️ PR merge time 3-4 weeks  
⚠️ 50-70% issues resolved  
⚠️ Code coverage 70-80%  
⚠️ 1-2 mentees advancing  
⚠️ Stars growing slowly  
⚠️ Release delayed 1-2 weeks  
⚠️ Mixed community sentiment  

### 🔴 Red Lights (Action Needed)

❌ < 2 new contributors/month  
❌ PR merge time > 4 weeks  
❌ < 50% issues resolved  
❌ Code coverage < 70%  
❌ No mentees advancing  
❌ Stars stagnant  
❌ Release delayed > 2 weeks  
❌ Negative community sentiment
