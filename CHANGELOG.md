# Changelog

## [1.1.0] - 2026-06-21

### Added
- Dark mode toggle with persistence
- Export notes as plain JSON or Markdown
- Import notes from JSON or Markdown files
- Share notes via encrypted URL links
- Bulk delete and bulk tag operations
- Toast notification system
- Keyboard shortcuts reference modal
- Comprehensive documentation (SECURITY.md, USER_GUIDE.md, CONTRIBUTING.md)

## [1.0.0] - 2026-06-01

### Added
- Note creation, editing, deletion
- AES-GCM-256 encryption per note
- Master password authentication (PBKDF2)
- Session auto-lock with configurable timeout
- Batch encrypt/decrypt operations
- Encrypted JSON export
- Biometric authentication (WebAuthn)
- Recovery codes (10 single-use)
- Audit trail
- Password strength meter
- GitHub Pages and Vercel deployments
