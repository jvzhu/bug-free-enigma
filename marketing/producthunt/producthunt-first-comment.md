## PRIMARY MAKER COMMENT (copy-paste ready)

```
Hey Product Hunt! 👋 I'm the maker of Bug-Free Enigma.

I built this because I was frustrated with cloud note apps that:
• Ask for your email before you can even try the product
• Sync data to servers you can't inspect or control
• Charge monthly fees for features you barely use
• Can (in theory) read your private notes

I wanted something completely private, genuinely secure, and free — forever.

So I built it. Here's what makes it different:

🔐 Real cryptography, not marketing copy
Every note is encrypted using AES-GCM-256 powered by your browser's native
Web Crypto API. Your master password is never stored — it's used on the fly
to derive an encryption key via PBKDF2 (100,000 iterations + HKDF). Even if
someone stole your device, they'd see only encrypted ciphertext.

🛡️ Features I didn't expect to build (but loved building):
• WebAuthn biometric login — fingerprint / Face ID in the browser
• Session auto-lock with a 2-minute visual warning (handles multi-tab too)
• 10 single-use recovery codes for account recovery without a backend
• Full audit trail of every security event
• Encrypted shareable links — share a note and only the recipient can decrypt

🆓 Free forever, open source
All features are free. The source code is on GitHub under MIT licence.
Inspect every line. Fork it. Self-host it.

🚀 No install, no account, no backend
Just open https://jvzhu.github.io/bug-free-enigma/ and start writing.
```
