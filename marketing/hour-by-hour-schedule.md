## HOUR-BY-HOUR RESPONSE TEMPLATES

### ProductHunt Comment — Thank You
```
Thank you so much [NAME]! Really appreciate the support. Let me know if
you have any questions about [TOPIC THEY MENTIONED] — happy to dig into
the details!
```

### ProductHunt Comment — Feature Request
```
Great suggestion! [FEATURE] is actually on our roadmap. I've added your
vote to it. If you want to follow along as we build it, GitHub Issues is
the best place: https://github.com/jvzhu/bug-free-enigma/issues
```

### ProductHunt Comment — Bug Report
```
Thanks for reporting this! I'm looking into it now. Can you share which
browser/OS you're on? I'll get a fix deployed ASAP and reply here when
it's live.
```

### ProductHunt Comment — Technical Question
```
Great question! [ANSWER]. If you want the full technical breakdown, I
wrote about it here: [LINK]. Happy to go deeper if you're curious!
```

### ProductHunt Comment — Comparison to Competitor
```
[COMPETITOR] is great for [USE CASE]. Bug-Free Enigma is different in
that [DIFFERENTIATOR]. If [YOUR KEY VALUE PROP] matters to you, I think
you'll find it worth trying. Let me know what you think!
```
